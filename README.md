# flutter_tiktok

仿写抖音的flutter app。主要实现了看视频功能

# 实现功能

- 上下刷视频，视频会自动加载封面
- 左右滑动去搜索与个人中心
- 双击冒爱心点赞
- 看评论
- 切换底部Tabbar
